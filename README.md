# Stingray

![Stingray](logo.png)

Stingray is a PoW-PoS-based cryptocurrency.

Block Spacing: `90 Seconds`

Stake Minimum Age: `1 Day`

Stake Maximum Age: `Unlimited`

Port: `33820`

RPC Port: `33821`
